# Boxis.io - service for creating website from your instagram profile. (Frontend)

## Stack (Frontend)
- ReactJS
- NextJS
- Bootstrap
- MaterialUI
- SCSS

## How to start
```
npm install
npm run dev
```

## How to build
```
npm run build
npm run start
```
